# G.S Ledger

**Controle financeiro automático**

G.S Ledger é um aplicativo Android de uso pessoal criado para registrar automaticamente entradas e saídas financeiras a partir de notificações bancárias.

## Objetivo
Eliminar a necessidade de anotações manuais, oferecendo controle financeiro simples, privado e automático.

## Funcionalidades (v1)
- Leitura de notificações
- Identificação automática de entradas e saídas
- Registro local (offline)
- Histórico financeiro
- Cálculo de saldo

## Estilo
- Azul moderno
- Visual sofisticado
- Interface simples e funcional

## Privacidade
Todos os dados ficam armazenados localmente no dispositivo.
